# TruthLens

AI-assisted media analysis Chrome extension for Instagram, TikTok, and Facebook.

## Features

- AI likelihood analysis
- Real-time overlay badges
- Local-first privacy architecture
- Lightweight Chrome extension
- Minimal permissions

## Privacy First

TruthLens is designed with minimal data collection and local-first processing.

No browsing history is collected.
No advertising tracking is used.
No user data is sold.

## Supported Platforms

- Instagram
- TikTok
- Facebook

## Disclaimer

TruthLens provides confidence estimates only and should not be treated as forensic verification.
